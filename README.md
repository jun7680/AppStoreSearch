# AppStoreSearch

### 기능구현 목록

1. 앱 id 검색 기능

   - 성공했고 result가 비어있지 않으면 디테일 화면으로 전환
   - 비어있을 경우 메시지 출력
   - 실패시 에러메시지 화면에 출력

2. 디테일 화면

   - 스크롤시 앱 아이콘 네비게이션 타이틀에 표시

   - 더보기 버튼으로 앱 설명 부분 확장

   - CompositionalLayout을 활용한 Section별 Layout정의

   - 스크린샷 전체 탐색 및 선택시 Preview형태로 전환
     전환시 이전 화면에서 선택된 스크린샷으로 초기화면 present
